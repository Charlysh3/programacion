Introducción a las estructuras de datos

1.1 Clasificación de las estructuras de datos

  • Primitivas: Datos básicos como enteros, flotantes, caracteres y booleanos.
  • No primitivas: Construidas a partir de primitivas.
  • Lineales: Arreglos, listas, pilas y colas.
  • No lineales: Árboles y grafos.
  • Estáticas: Tamaño fijo, como arreglos tradicionales.
  • Dinámicas: Tamaño variable, como listas enlazadas.

1.2 Tipos de datos abstractos (TDA)

  • TDA: Representación lógica de datos y sus operaciones.
  • Listas: Acceso por índice.
  • Pilas: Operaciones en el tope.
  • Colas: Orden de llegada y salida.

1.3 Ejemplos de TDA's

  • Colas: Gestión de procesos en sistemas operativos.
  • Pilas: Manejo de expresiones aritméticas.

1.4 Manejo de memoria

  • Memoria estática: Asignada al inicio, usada por variables globales/locales.
  • Memoria dinámica: Asignada/liberada durante la ejecución. Java usa recolector de basura.

1.5 Análisis de algoritmos

  • Complejidad temporal: Tiempo de ejecución según el tamaño de entrada (Big-O).
  • Complejidad espacial: Memoria requerida para ejecutar el algoritmo.
  • Eficiencia: Evalúa tiempo y memoria utilizados en comparación con otros algoritmos.
