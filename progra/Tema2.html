Recursividad

¿Qué es la Recursividad?

La recursividad en programación es un concepto que consiste en que una función se llama a sí misma para resolver un problema. Cada vez que una función se llama a sí misma, se produce un paso de recursión.

¿Cuándo usar la recursividad?

La recursividad es útil para resolver problemas que se pueden definir en sus propios términos. Por ejemplo, para recorrer un árbol de elementos, como un sistema de archivos o el árbol de elementos de una página web, se puede utilizar la recursividad.

Ventajas y desventajas

  • Ventajas: Permite abordar problemas complejos de manera simple y elegante.
  • Desventajas: Los algoritmos recursivos pueden ser más lentos y consumir más recursos al ejecutarse en un ordenador.

Ejemplo práctico de recursividad

Veamos un ejemplo de cómo se puede usar la recursividad para calcular el factorial de un número.

Ejemplo en pseudocódigo:

función factorial(n):
    si n == 0:
        devolver 1
    sino:
        devolver n * factorial(n - 1)

resultado = factorial(5)  // Llamada recursiva: 5 * 4 * 3 * 2 * 1
imprimir(resultado)  // Imprime 120
